#  STRYDE — Premium Footwear E-Commerce Landing Page

A modern, luxury-style **E-Commerce Landing Page** built using **HTML, CSS, and JavaScript**.  
This project showcases a complete frontend shopping experience with a premium UI design, interactive cart system, and checkout flow.

##  Project Overview

**STRYDE** is a premium footwear landing page designed to deliver a high-end shopping experience.

This project includes:
- Luxury UI inspired by modern brands  
- Fully responsive layout  
- Interactive shopping cart system  
- Checkout and order simulation  
- Smooth animations and transitions  

It is a **frontend-only project**, built without backend integration, making it ideal for portfolio showcasing.

---

##  Key Features

###  Modern UI/UX Design
- Clean and minimal interface  
- Typography using Google Fonts (Playfair + Inter)  
- Smooth hover effects and transitions  

###  Product System
- Dynamic product rendering using JavaScript  
- Product cards with images, prices, and discounts  
- “Add to Cart” and “Shop Now” functionality  

###  Shopping Cart Sidebar
- Slide-in cart panel  
- Real-time cart updates  
- Remove items from cart  
- Total price calculation  

###  Checkout System
- Order form with:
  - Name  
  - Email  
  - Address  
  - Payment method  
- Payment simulation (COD, Card, JazzCash, Easypaisa)  

###  Toast Notifications
- Add/remove cart feedback  
- Order success messages  
- Newsletter subscription feedback  

###  Fully Responsive
- Mobile-friendly design  
- Adaptive layout for all devices  

---

##  How It Works

1. Product data is stored inside a JavaScript array  
2. Products are dynamically rendered into the UI  
3. Users can:
   - Add items to cart  
   - Remove items  
   - Open checkout modal  
4. Cart updates in real-time  
5. Checkout form simulates order placement  
6. Toast messages provide user feedback
